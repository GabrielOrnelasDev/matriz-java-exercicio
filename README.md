# matriz-java-exercicio
Sobre o projeto

Este projeto foi desenvolvido para praticar o uso de matrizes (arrays bidimensionais) em Java.

O programa permite:

Ler uma matriz de tamanho definido pelo usuário
Exibir os valores da matriz
Buscar um número específico
Mostrar os valores vizinhos (esquerda, direita, cima e baixo)
